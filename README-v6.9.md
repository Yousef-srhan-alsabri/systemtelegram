# v6.9 — UX, Campaign Planner & Search/Join Engine

هذا الإصدار يركز على تنظيم تجربة الاستخدام وإعادة تصميم الأجزاء التي أصبحت كبيرة:

## الواجهات والتصميم
- إزالة نافذة الحملات الثابتة التي كانت تحجب الصفحات.
- استبدالها بزر عائم صغير يفتح لوحة جانبية Offcanvas عند الطلب.
- تحسين responsive للجوال واللابتوب.
- تحسين cards والجداول وخطوات إنشاء الحملة.

## إنشاء الحملات
- مخطط حملة متعدد المراحل.
- دعم محتوى موحد لكل الحسابات.
- دعم محتوى مختلف لكل حساب.
- دعم تحويل آخر رسالة لكل حساب أو بشكل موحد.
- جدولة كل حساب بوقت مستقل.
- توقيت افتراضي Asia/Aden.
- إدخال الوقت بنظام 12 ساعة: صباحاً / مساءً.

## البحث
- استثناء قنوات النظام تلقائياً من البحث العام: قناة تصدير الروابط ومصادر الانضمام.
- توسيع ذكي لعبارة البحث بمرادفات وصيغ قريبة.
- عدد نتائج قابل للاختيار: 50 / 100 / 200 / 500 / 1000.
- استمرار دعم البحث العام، داخل الحسابات، أو الاثنين معاً.

## الانضمام
- إضافة أزرار تحكم في تفاصيل مهمة الانضمام: إيقاف، استئناف، إلغاء، إعادة محاولة الفاشل.
- استمرار دعم الاستئناف الديناميكي بعد FloodWait عبر scheduler_worker.py.
- تفاصيل أوضح لوقت الاستئناف وحدود Telegram.

## ملاحظات تشغيل
لتفعيل الجدولة واستئناف الانضمام بعد حدود Telegram، شغّل:

```bash
python scheduler_worker.py
```

ولتشغيل الويب محلياً:

```bash
python run.py
```

على Railway استخدم نفس إعدادات v6.8:

```bash
gunicorn wsgi:app --bind 0.0.0.0:$PORT --workers 2 --threads 4 --timeout 180
```

وخدمة مستقلة للمجدول:

```bash
python scheduler_worker.py
```
